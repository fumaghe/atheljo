What can you do with Avalon?

Avalon allows you to manage and monitor the telemetry data in real time, generate custom reports, and analyze critical metrics on an intuitive platform.

Avalon offers three subscription tiers (Essential, Advantage, and Premiere) with varying costs based on the features required.

Does it support real-time data?

Yes, Avalon supports real-time data access for immediate monitoring.

What happens to Merlin?

MerlinDB will be used as the main data storage for the telemetry.

What happens to Archimedes?

Archimedes has been updated and changed. Right now we are still implementing all the older features.

How do Merlin and Archimedes work with Avalon?

Merlin will be main source to retrieve telemetry data and metadata. Archimedes then adapts the data and makes the necessary analysis and stores them in a Firestore database.

Why do you need the Firestore database?

Firestore is a noSQL document based database that allows Avalon to get data faster and store it for long-time analysis. Furthermore, it is managed by Google, so after the migration to AlloyDB, it will be possible to retrieve and analize faster.

Why did you choose the name Avalon?
